# Glossary

## MVP
Минимальная учебная версия, которая запускается и демонстрирует основные экраны.

## Future
Идеи “на потом”, которые требуют backend/внешних сервисов.

## SPA
Single Page Application — навигация без полной перезагрузки страницы.

## Hash‑роутинг
Роутинг через `location.hash` (пример: `/#/vacancies`). Подходит для статического хостинга.

## API (в рамках проекта)
`utils/api.js` — слой работы с `localStorage`, не настоящий сервер.

## Store
Состояние приложения (текущий пользователь, настройки UI).
