# Roadmap

## Легенда
- **MVP** — обязательно для учебной версии (без сервера/БД).
- **Future** — идеи “на потом”, требующие backend/внешних сервисов.

## Критический путь (MVP)
1. Каркас проекта (структура + `index.html` + стили + `main/router/store`).
2. Hash‑роутинг + базовые страницы (home, 404).
3. `utils/api.js` + модель `localStorage` (по `DATA_MODEL.md`).
4. Аутентификация localStorage-only (register/login/logout + guard).
5. Каталоги вакансий/проектов + детали.
6. Профиль.
7. Полировка и проверка по `QA.md`.
