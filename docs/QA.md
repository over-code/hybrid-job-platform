# QA (ручная проверка)

## 1) Запуск
- [ ] Live Server открывает проект без пустого экрана.
- [ ] В консоли нет критических ошибок.

## 2) Роутинг
- [ ] Переходы по меню работают без перезагрузки.
- [ ] Прямая вставка `/#/route` открывает правильную страницу.
- [ ] Несуществующий маршрут показывает 404.

## 3) Аутентификация (localStorage)
- [ ] Регистрация создаёт пользователя в `hjp:users`.
- [ ] Вход устанавливает `hjp:session.currentUserId`.
- [ ] Выход очищает сессию.
- [ ] Приватные страницы недоступны без входа.

## 4) Вакансии
- [ ] Список вакансий рендерится из `hjp:vacancies`.
- [ ] Фильтры меняют список.
- [ ] Детали вакансии открываются.

## 5) Проекты
- [ ] Список проектов рендерится из `hjp:projects`.
- [ ] Детали проекта открываются.

## 6) Профиль
- [ ] Профиль показывает данные текущего пользователя.
- [ ] Изменения сохраняются в `hjp:users`.

## 7) UI
- [ ] Ок на 360px и 1280px.
- [ ] Элементы управления доступны с клавиатуры (минимум: tab).
