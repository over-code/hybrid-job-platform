# Документация (hybrid-job-platform)

Учебный фронтенд‑прототип (HTML/CSS/JS) без сервера и без базы данных.

## Быстрая навигация
- `OVERVIEW.md` — что делаем (scope), для кого, что входит в MVP
- `ARCHITECTURE.md` — архитектура и структура проекта
- `DATA_MODEL.md` — модель данных в `localStorage`
- `ROADMAP.md` — план работ и приоритеты
- `QA.md` — чек‑лист ручной проверки
- `GLOSSARY.md` — термины и договорённости

## Главное правило
Любая функциональность должна работать **в статическом запуске через Live Server**.
Если для реализации нужен backend — это **не MVP** и идёт в Future.
