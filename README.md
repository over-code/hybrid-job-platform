# hybrid-job-platform

Учебный проект для колледжа: фронтенд‑прототип платформы вакансий и фриланса.

## Ограничения
- Без сервера и без базы данных.
- Данные хранятся в `localStorage`.

## Запуск
- Открой папку проекта в IDE.
- Запусти Live Server ("Go Live").
- Открой `http://localhost:5500`.
