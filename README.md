# Hybrid Job Platform

[![Статус](https://img.shields.io/badge/статус-завершено-success)](https://github.com/over-code/hybrid-job-platform)

Учебный проект для колледжа: фронтенд-прототип платформы для поиска работы и фриланс-проектов.

## Возможности

- Аутентификация пользователей (регистрация, вход, выход)
- Просмотр и фильтрация вакансий
- Просмотр и фильтрация фриланс-проектов
- Личный кабинет пользователя
- Адаптивный дизайн для всех устройств

## Технические детали

- Чистый JavaScript (ES6+)
- Хранение данных в `localStorage`
- SPA-архитектура с хэш-роутингом
- Адаптивная верстка (мобильные и десктопные устройства)

## Запуск

1. Клонируйте репозиторий
2. Откройте папку проекта в IDE
3. Запустите Live Server (расширение "Live Server" для VS Code)
4. Откройте `http://localhost:5500` в браузере
